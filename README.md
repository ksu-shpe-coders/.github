# SHPE Coders @ KSU

The software development team within the Society of Hispanic Professional Engineers (SHPE) chapter at Kennesaw State University. We build tools and applications that serve our club and community, while learning real-world engineering skills.

## What we're building

### 📱 [shpe-mobile](https://github.com/shpe-ksu-coders/shpe-mobile)
The official SHPE KSU mobile app. Built with React Native and Expo. Features include event management, member directory, push notifications, and admin tools.

### ⚙️ [shpe-api](https://github.com/shpe-ksu-coders/shpe-api)
The backend API that powers our applications. Built with ASP.NET Core 10 and PostgreSQL. Designed to serve both the mobile app and future web tools.

### 🌐 shpe-web (planned)
Web companion to the mobile app — admin dashboard and public-facing chapter site. Planned for Fall 2026 after the mobile beta ships.

## Tech stack

- **Backend:** ASP.NET Core 10, PostgreSQL, Entity Framework Core
- **Mobile:** React Native, Expo, TypeScript
- **Notifications:** Expo Push Service
- **Hosting:** Azure for Students (development), KSU CCSE servers (production target)

## Status

In active development. Beta target: end of August 2026 for eboard testing, broader rollout in Fall 2026.
