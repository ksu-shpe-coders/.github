# SHPE Coders @ KSU

The software development team within the Society of Hispanic Professional Engineers (SHPE) chapter at Kennesaw State University. We build tools and applications that serve our club and community, while learning real-world engineering skills.

## What we're building

### 🚀 [shpe-mobile](https://github.com/shpe-ksu-coders/shpe-mobile)
The official SHPE KSU mobile app. Built with React Native and Expo. Features include event management, member directory, push notifications, and in-app messaging.

### ⚙️ [shpe-api](https://github.com/shpe-ksu-coders/shpe-api)
The backend API that powers our applications. Built with ASP.NET Core 10 and PostgreSQL. Serves both the mobile app and future web tools.

### 🌐 [shpe-web](https://github.com/shpe-ksu-coders/shpe-web)
(Coming soon) Web companion to the mobile app — admin dashboard and public-facing chapter site.
